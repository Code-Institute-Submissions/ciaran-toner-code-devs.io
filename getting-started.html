Getting started on your codeing journey has never been easier.

Learning to code can seem like a long arduous journey. However, don’t let this stop you! Over the last decade there has been a massive jump in the demand in I.T & programming, Everything from machine learning to automatising our everyday tasks to building blogs. There has never been a better time to learn to code.

What you need to get started.

First you would need a text editor to help you sort and save your files. Our two top pics for beginners would be Atom text editor & Sublime text editor, both are good choices. All websites contain three main elements that make up a webpage, HTML, CSS & Javascript, But for the time being were going to focus just on HTML.

Your first webpage!

There are 3 main elements needed for your webpage that tells the browser how your webpage is formatted and how to correctly display everything. You can find the basics on how to set up your first page right here.All of your web pages should be saved as a .html page e.g home-page.html. This tell the browser that your page is a html document, you can find more on why you save your webpages with a .html tag here

Whats next?

After you learned the basics on the structure of the HTML document and the main elements you will use you can move on to CSS & then Javascript. Just remember learning HTML & CSS can take some time, so take your time & remember there is fantastic resources out there to help you become a great web developer. You there are three great free resources worth mentioning. W3schools, Freecodecamp, Stackoverflow & many more!

wrapping things up

While this article is just scratches the surface on how to become a great web developer don’t worry about how much information is out there. The important message is to just start! Never be afraid to ask for help, many expierienced developers that where in the same position you are in. The key to being a great developer is the willings to learn, problem solve, ask or help when you might need it & enjoy every step on becoming a developer.
